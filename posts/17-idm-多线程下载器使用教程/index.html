17-IDM多线程下载器使用教程

一说到下载器，大家的第一想到的可能就是迅雷、QQ旋风、快车之类的下载软件了，现在0202年都快结束了，你不能不知道的一款 windows 下载神器！

IDM (Internet Download Manager) 是商用的付费软件，Chrome和FireFox有扩展支持。支持多媒体下载、自动捕获链接、自动识别文件名、静默下载、批量下载、计划下载任务、站点抓取、队列与网盘支持等，主打“稳定”、“多文件管理”，功能很齐全，是老牌下载器之一。

下载 IDM 安装包

  • 官方下载 ：http://www.internetdownloadmanager.com/download.html 下载后可以免费试用 30天

  • 数码荔枝：99一年授权，149永久授权。

  • 第三方下载：本教程以安装 第三方修改一键安装版 IDM_6.38.7_俄版 为例，演示安装及其使用技巧 ~

安装软件

将软件安装包解压后，双击应用程序安装，此处使用的解压软件为 Bandzip，其他解压软件同理，注意这里是解压不是预览压缩包。

安装过程中电脑屏幕会闪动几下，这是正常现象。安装完成后会在桌面创建软件图标，并且在任务栏右下角你可以看到软件的缩略图标。至此，软件安装完毕，安装完后可以将软件压缩包以及解压后产生的文件夹删除。

软件的基本优化

  • 更改语言

软件安装完毕后，你会发现软件的语言是英文，哈哈，因为这是俄罗斯大神修改版，咱们直接改个语言就行了。 View --> Language --> 简体中文

  • 更改下载线程（连接数）

这里的最大连接数不是说设置的越大越好，这个要看网站对用户的发送的文件请求是否有限制。有一些版权意识较强的网站会限制用户的文件请求，通常在单次下载文件过程中只能发送一次文件请求，并且不支持断点续传。也就是说，就算你线程数设置的数目对下载文件没有任何影响，有时候反而会起副作用。

开启拓展程序

如果你使用的是系统自带的Chromium内核 的 Edge浏览器，你需要在设置 –> 拓展 –> 点击开启拓展

这里推荐使用 Chrome 浏览器，安装IDM后，会自动给Chrome 浏览器安装并开启该拓展

文件下载测试

  • 视频下载：爱奇艺、腾讯视频、优酷、Bilbili，youtube等等

只要你打开想要下载的音视频页面，IDM 会自动检测在线播放器发出的多媒体请求并在播放器上显示下载浮动条，你可以直接下载流媒体网站中的视频进行离线观看。支持 MP4、MP3、MOV、AAC 等常见音视频格式的检测与下载。

  • 下载文件测试

​ 当弹出下载窗口时，IDM已经开始在后台开始下载了，这会让你产生一种错觉 → 点击开始下载后，你发现文件已经下载了一大半 ~~ 这是因为 IDM 下载时的连接数很多，下载界面显示传输速度已经达到了 11.7Mbps，快要拉满 100M带宽了。细心的你会发现下面下载进度条中有16根小红线，这正是我们之前设置的最大连接数，如果你修改成了32，那么下方将会显示32根小红线，并且更加密集。增加下载线程会将你的下载文件速度最高提升 5~6 倍 ！

  • 下载特殊的 m3u8文件

此处的m3u8文件实质上是一个播放列表，意思是将一个完整的视频分割成一小段一小段，然后逐个放给你看。有些网站的m3u8文件是可以直接利用IDM下载的，并且能够下载到完整的视频，但拿国内的腾讯课堂来说，这个网站对视频文件进行了分割，我们需要下载所有的视频片段进行拼接，这给我们下载一个完整的视频带来很大的麻烦，我们可以通过 IDM + N_m3u8DL-CLI.exe 的方式解决，具体内容请参考往期文章：https://betterman.xyz/348.html

卸载

从安装到卸载，一条龙服务！

Win + S 搜索控制面板，进入卸载软件，右键 IDM 即可卸载

注意，点击 卸载 后可能会重启电脑，请保存好当前工作环境并关闭所有软件后再进行卸载操作。

下载百度网盘文件

众所周知，pandownload事件发生后，大家都在骂百度。害，毕竟百度也是要恰饭的，但收费确实有点点贵了。那么作为一个普通用户该如何使用 IDM 来下载百度网盘的文件呢？

pandownload 软件内部使用的是 Aria2 多线程下载，其原理与 IDM 类似！

  • 具体步骤
  1. 选项 -> 下载 -> 手动添加任务时使用的用户代理（UA）-> 填入 LogStatistic
  2. 右键复制下载链接（直接点击 或 右键调用 IDM 将传入浏览器的 UA，将会导致下载失败），在 IDM 新建任务，粘贴链接即可下载。
  3. 具体内容请查看往期文章《最新福利–百度网盘在线解析满速下载》
